## About DEVFORGE

**DEVFORGE** is a creative tech brand and portfolio platform engineered from scratch as an independent passion project. It showcases original UI engineering pushed to its limits — immersive animations, a coherent neon design system, and a visual identity strong enough to stand as a product on its own.

DEVFORGE reflects a commitment to **intentional design** and **engineering craftsmanship** beyond client work.

---

## Design Language

| Element | Value |
|---------|-------|
| Primary Accent | Neon Blue `#00CFFF` |
| Aesthetic | Cyberpunk · Dark Tech · Futuristic |
| Typography | Display-driven with monospace technical accents |
| Motion | Canvas-based animation · CSS keyframes · 60fps target |

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React · Vite |
| Animation | HTML5 Canvas API · CSS Animations |
| Styling | Custom CSS Design System |
| Language | JavaScript · HTML · CSS |

---

## Engineering Highlights

- **Canvas-based animation engine** running custom particle systems and intro sequences at 60fps
- **Advanced CSS keyframes** orchestrating multi-layer reveal animations
- **Coherent dark design system** with strict palette discipline (neon blue on near-black)
- **Component-driven architecture** built on React + Vite for fast iteration
- **Original brand identity** — every visual element designed and engineered from scratch

---

## Local Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

No environment variables required — DEVFORGE is fully self-contained.

---

## Security Practices

- No secrets, credentials, or API keys are present in this repository
- `.gitignore` excludes build artifacts and local development files
- All third-party assets used are properly licensed or original creations
